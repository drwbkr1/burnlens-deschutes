# BurnLens Prompt Log Navigation

## Purpose

This root file is a navigation and compatibility entry point for BurnLens prompt/build logging.

It is not the canonical protocol, the canonical entry template, a second task index, a status register, or a transcript archive.

## Canonical sources

- [Prompt/build-log protocol and entry index](records/PROMPT_BUILD_LOG.md)
- [Detailed prompt-log entry template](templates/PROMPT_LOG_ENTRY.md)

## Minimal use sequence

1. Read the canonical protocol and entry index.
2. Copy the canonical detailed entry template.
3. Save the completed dated entry under `records/prompt-build-log/`.
4. Update the canonical index when the protocol requires it.

## No-duplicate-source rule

Keep logging rules, required fields, entry status, and task indexing in the canonical files above. Do not reproduce them here.
