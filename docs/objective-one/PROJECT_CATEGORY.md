# BurnLens Deschutes — Project Category

## Locked category

**BurnLens Deschutes is a computer vision and geospatial intelligence portfolio project.**

## Category statement

BurnLens Deschutes is a CV + GEOINT portfolio project for wildfire-related screening in Deschutes County, Oregon. It demonstrates a reproducible workflow for turning imagery, public geospatial layers, and experimental segmentation or baseline outputs into traceable maps and exposure-style summaries.

## Primary classification

BurnLens Deschutes should be classified as:

- **Computer vision** because it is designed around image-derived masks and future segmentation outputs.
- **Geospatial intelligence / GEOINT** because the outputs are spatial, map-ready, and connected to public geospatial layers.
- **Portfolio project** because its purpose is to demonstrate technical capability, not to operate as an official system.
- **Wildfire screening workflow** because it organizes wildfire-relevant imagery and context into experimental screening artifacts.

## What this category enables

This category allows the build to include:

- imagery preprocessing
- segmentation or baseline-mask generation
- raster and vector outputs
- geospatial overlays
- run manifests
- map exports
- exposure-style summaries
- model cards and source documentation
- website presentation through burnlensproject.org

## What this category prevents

This category prevents scope drift into:

- live emergency response
- evacuation routing or instructions
- official hazard mapping
- field-validated wildfire modeling
- county, state, federal, or fire-service authority
- utility-grade operational analysis
- broad wildfire platform claims
