# BurnLens Deschutes — Project Thesis

## Primary thesis

**BurnLens Deschutes demonstrates how computer vision and public geospatial data can be combined into a transparent wildfire GEOINT workflow that produces traceable maps, exposure-style summaries, and clearly documented limitations for a defined Deschutes County study area.**

## Short version

BurnLens Deschutes turns wildfire-relevant imagery and public geospatial layers into experimental, map-ready screening outputs with traceable methods and honest limits.

## Portfolio value proposition

BurnLens Deschutes is designed to show more than a model output. It demonstrates an end-to-end workflow:

**source data → preprocessing → segmentation or baseline output → geospatial overlay → exposure-style summary → documented run package**

The project should show technical capability in computer vision, geospatial processing, version control, output packaging, and responsible communication.

## What the thesis does not claim

This thesis does not claim that BurnLens Deschutes is official, operational, field validated, agency endorsed, or suitable for emergency decisions. It frames the work as a transparent portfolio demonstration.
