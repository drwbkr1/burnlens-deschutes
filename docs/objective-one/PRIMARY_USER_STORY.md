# BurnLens Deschutes — Primary User Story

## Primary user story

**As a technical or planning-adjacent portfolio reviewer, I want to inspect a reproducible wildfire GEOINT workflow that combines imagery, public geospatial layers, and experimental computer vision outputs, so I can evaluate whether BurnLens Deschutes demonstrates useful, transparent, and responsibly bounded wildfire screening capability for a defined Deschutes County study area.**

## Supporting story

As a reviewer, I want each map, summary, and screenshot to trace back to a run ID, source metadata, versioned data, and method notes, so I can distinguish a reproducible workflow from a one-off visualization.

## Success conditions

The user story is satisfied when a reviewer can answer:

- What geography is being studied?
- What data sources were used?
- What method produced the output?
- What map or summary was generated?
- What version and run ID produced it?
- What limitations apply?
- What official information would supersede it?

## Boundary

This user story does not imply that BurnLens Deschutes supports emergency response, evacuation routing, official hazard mapping, field validation, or agency-endorsed decision-making.
