# Making Independent Review Executable Without Pretending It Happened

BurnLens began this cycle with a stable three-event proposal but no honest way for another person to review it. The software could reproduce every pixel; that still said nothing about whether an independent analyst agreed with the labels.

The first design temptation was to score the proposal ourselves. That would have produced a polished number with no independent evidence behind it. The bounded checkpoint instead builds the missing instrument.

The new packet reopens the exact registered source packages and recomputes all three proposal arrays before selecting a sample. Four units are drawn from every proposal state present in every event, using immutable hash ranking and preferred spatial separation. McKay has no excluded pixels, so its excluded stratum stays empty. The final 56 units cover 14 present strata.

Presentation order uses a second deterministic hash so state and event groups are not shown as contiguous blocks. Each first-pass card shows pre/post Sentinel imagery, continuous dNBR, and NIFC or MTBS context. BurnLens's proposed state and binary target are withheld until a separate reveal. The first-pass material is proposal-blinded in content and ordering, but the repository does not overclaim cryptographic blindness or controlled study conditions.

The response template contains no proposal-value fields. A separate verifier rejects altered packet bindings, duplicate or invalid samples, target promotion of ignored states, reveal tokens in the blind page, incomplete templates, out-of-domain responses, missing independence attestations, and invalid timestamp order. With no response files supplied, it can only report packet integrity and review readiness.

That distinction is the checkpoint's value. BurnLens now has a real, reproducible path to independent label evidence, while the dataset remains honestly deferred.

> Experimental BurnLens CV evidence. Not official wildfire information. Not emergency guidance. Not evacuation, routing, tactical, or incident-command support. Official sources govern.
