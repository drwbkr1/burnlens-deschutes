# August 6 submission-bundle decision

**Issue / milestone:** #544 / `BL-PORT-002 / P2O4-T37`

**Decision:** `PACKAGE_VERIFIED_LOCAL_REVIEWER_PATH_WITHOUT_EXPANDING_CLAIMS`

BurnLens 0.48.0 proved the reviewer experience. The remaining deadline risk was
handoff friction: a reviewer still needed the repository layout and had no
single extracted start file.

P2O4-T37 packages only the exact verified public roster. After extraction, a
reviewer opens `START-HERE.html`. The rewritten page keeps the same evidence,
design, source roles, limitations, and lineage while resolving every link
inside the bundle.

The ZIP is deterministic, no-overwrite, safely structured, internally
manifested, externally receipted, and independently extractable. It contains
no credentials, private custody, provider bytes, tracking, upload, executable
launcher, deployment, or public-sharing change.

Dataset, split, baseline, and model remain null. This packaging decision does
not change scientific evidence, labels, the project promise, target user, CV
task, phase outcomes, source precedence, use boundaries, or claims.
