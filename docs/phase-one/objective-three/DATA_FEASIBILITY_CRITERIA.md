# Phase 1 / Objective Three — Data Feasibility Criteria

Draft created after branch creation and fresh source research.

This document is documentation-only. It defines criteria for judging candidate data sources before future implementation.

## Criteria

1. Source authority.
2. Access path reproducibility.
3. Spatial resolution.
4. Temporal resolution and latency.
5. Deschutes County coverage.
6. Band and product suitability.
7. Quality and uncertainty handling.
8. Segmentation suitability.
9. Reference suitability.
10. Baseline suitability.
11. Overlay suitability.
12. Format and CRS compatibility.
13. Citation and attribution requirements.
14. Source-precedence compatibility.
15. Provenance readiness.
