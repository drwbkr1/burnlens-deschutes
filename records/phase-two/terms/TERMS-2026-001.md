# TERMS-2026-001 — Source Terms and Access Decision

**Checked:** 2026-07-13

**Scope:** Metadata retention and future-role feasibility only

## Decision table

| Source | Primary-source evidence | Decision for this checkpoint | Future constraint |
|---|---|---|---|
| Copernicus Sentinel-2 data | Sentinel Data Legal Notice permits free, full, and open access and allows reproduction, distribution, communication, adaptation, and modification with attribution | Retain normalized metadata; accepted | Use `Copernicus Sentinel data [Year]` or `Contains modified Copernicus Sentinel data [Year]`; recheck actual asset endpoint and quotas |
| CDSE portal/catalog | CDSE terms distinguish Sentinel data from other portal content and state that registration is normally needed to access EO data | Public STAC metadata query accepted; portal media not copied | Do not infer that every asset endpoint is anonymous; do not redistribute protected portal content |
| NASA VIIRS data and CMR metadata | NASA Earthdata says NASA-led data are CC0 unless marked and asks users to acknowledge/cite; product catalog says the datasets are openly shared without restriction | Retain normalized metadata; accepted | Cite the exact dataset DOI and access date when assets or derived results are used |
| FIRMS archive/API | Archive requires Earthdata or emailed authentication; API requires a map key | Deferred; no access attempted | Owner approval is required before adding a secret or account-backed route |
| Oregon OSFM release | Official factual incident statement | Link and paraphrase limited facts; accepted | Do not reuse page media or treat prose as a spatial dataset |
| U.S. Census representative point | Official Census geographic reference | Retain coordinate as AOI rationale; accepted | Treat it as a representative point, not a precise incident location |

## Sentinel attribution requirement

Future output using 2024 Sentinel data must carry the applicable notice:

- `Copernicus Sentinel data 2024`, or
- `Contains modified Copernicus Sentinel data 2024`.

## NASA citation requirement

Future use must identify the exact collection and version, applicable DOI, provider, and access date. This checkpoint cites source metadata but does not publish VIIRS data or a derived result.

## Unresolved matters that do not block metadata retention

- Anonymous versus authenticated download behavior for the exact Sentinel band assets was not tested.
- NASA CMR metadata is public, while actual LP DAAC asset retrieval may require Earthdata Login.
- Redistribution and storage behavior must be rechecked for the actual asset endpoint and derivative to be created.

These items block asset intake, not the no-secret public metadata queries completed here.
